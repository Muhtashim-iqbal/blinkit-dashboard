# BlinkIT Grocery Sales Dashboard

## Project Overview

This project analyzes BlinkIT grocery sales data using Power BI and SQL.

## Tools Used

* Power BI
* SQL
* Excel

## Features

* Sales performance analysis
* KPI dashboard
* Category-wise analysis
* Outlet performance insights

## Dataset

The dataset contains grocery sales transaction details.

## Files Included

* Power BI dashboard (.pbix)
* Excel dataset
* SQL analysis files
* Dashboard images
